# Walkthrough: ER図の更新

## 手順
1. `docs/images/er_diagram.mmd` の作成
   - 既存のPythonスクリプトからテーブル構造とリレーションを抽出
   - Mermaid構文へ変換
   - クラス定義による色分けを追加

2. 画像生成 (`er_diagram.png`)
   - Mermaidのレンダリング用HTML作成
   - ブラウザオートメーションによるキャプチャ

3. クリーンアップ
   - 一時ファイルの削除

## 確認事項
- [ ] エンティティが正しいグループ色で表示されているか
- [ ] 重要なリレーションが結ばれているか
- [ ] PNG画像が生成されているか
