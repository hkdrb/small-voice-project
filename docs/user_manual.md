# 📘 Small Voice システム利用マニュアル

本マニュアルは、組織内ブロードリスニングシステム「Small Voice」の操作方法を解説するものです。
本システムは、日々の業務における気づきや改善案、システムに関するフィードバックなどの「小さな声」を収集し、AI分析を通じて組織改善に活かすことを目的としています。

## 🏠 2. 共通機能
すべてのユーザーが利用可能な基本機能です。

### 🔑 2.1 ログイン
1.  **アカウント取得**: アカウントは**システム管理者**によって作成・招待されます。招待メールのリンクまたは配布された情報を使用します。
2.  **ログイン**: ログイン画面にアクセスし、情報を入力してログインします。

### 📊 2.2 ダッシュボードの閲覧
ログイン直後に表示される画面です。サイドバー（モバイルの場合はメニュー）から各機能にアクセスできます。

- 📝 **アンケート回答**: 回答受付中のフォーム一覧が表示されます。
- 📑 **レポート閲覧**: 公開された分析セッションの一覧が表示されます。
- 🗣️ **雑談掲示板**: 組織内の「みんなの声」を共有する掲示板です。

### 🗣️ 2.3 雑談掲示板
日々の業務での気づきや、ふとした思いつきを気軽に投稿できる場所です。
※ 画面上の表記は「雑談掲示板」または「みんなの声」と表示されます。

- **投稿**: 画面右下の **`＋`** ボタン（または投稿ボタン）から新規投稿ができます。
- **返信**: 💬 アイコンをクリックして返信できます。
- **いいね**: ❤️ アイコンをクリックしてリアクションできます。
- **作成フォームの提案（AI分析）**: 管理者はここでの会話データをAI分析し、新しいアンケートの種などを発見します。

### 💡 2.4 分析セッションの閲覧と議論参加
公開された分析セッション（レポート）を確認し、課題ごとの議論に参加できます。
※ 公開済みのセッションのみ閲覧可能です。

1.  **レポート閲覧**: サイドバーの `レポート閲覧` から、見たい分析セッションをクリックします。
2.  **クラスタリング (回答分布)**: AIが回答を「意味」ごとに分類・配置したマップです。
    - 地図上の点をクリックすると、実際の回答内容が表示されます。
    - 似た意見が集まり、外れ値（特異点）が可視化されています。
3.  **課題リスト (アジェンダ)**:
    - **マジョリティ課題**: 多数派の意見から抽出された、取り組むべき主要な課題。
    - **Small Voice (少数意見)**: 埋もれがちだが重要な気づきを含んだ意見を、独立した課題としてピックアップしています。
4.  **AIファシリテーター**: 議論の進行状況をAIが分析し、「次のアクション」を提案した結果を閲覧できます。
    > [!NOTE]
    > 管理者が公開設定にしている場合のみ表示されます。
5.  **チャット参加**: 画面下部の「この課題についての議論」にて、意見投稿、リプライ、いいねが可能です。
    > [!TIP]
    > 自身のコメントは ✏️ **編集ボタン** から修正できます。

### 🔔 2.5 通知機能
システム内で重要なイベントが発生した際、ダッシュボード右上のヘッダーにある **🔔 ベルアイコン** に通知が届きます。

#### 基本ルール
- **自分除外**: 自分自身の操作（コメント投稿、レポート公開など）による通知は、自身には届きません。
- **自動遷移**: 分析セッション(課題)関連の通知をクリックすると、ページ遷移と同時に**該当する議論スレッドが自動的に開きます**。

#### 通知の種類と対象者

| カテゴリ | 内容 | 公開状態による制限 | 対象者 |
| :--- | :--- | :--- | :--- |
| **📢 全体のお知らせ** | フォーム公開、レポート公開、AI分析更新 | **公開済み**のもの限定 | 全メンバー |
| **🗣️ みんなの雑談** | 雑談掲示板への新規投稿・返信 | 常に通知 | 全メンバー |
| **✨ 雑談の分析** | 雑談データのAI分析による提案作成 | 管理者のみ（公開後は全員） | 管理者（公開時は全員） |
| **💬 レポート議論** | 課題ごとのチャット投稿、AI分析更新 | **公開中**なら全員、**未公開**なら管理者 | 条件による |
| **📩 申請のやり取り** | フォーム申請に対する承認・却下・返信 | 常に通知 | 申請者のみ |
| **🛡️ 管理業務** | 一般ユーザーからの新規申請・再申請 | 常に通知 | 全ての管理者 |

### ⚙️ 2.6 プロフィール設定
サイドバー最下部の `プロフィール設定` ボタンから、以下の情報を変更できます。
- **名前**: システム内での表示名。
- **パスワード**: ログインパスワード（変更する場合のみ入力）。

## 👤 3. 一般ユーザー
組織の一般メンバー向けの機能です。

### 📝 3.1 フォームへの回答
1.  ダッシュボードの `アンケート回答` タブ（または回答受付中エリア）を確認します。
2.  回答したいフォームを選んで **`回答する`** ボタンを押し、設問に回答して送信します。
    > [!NOTE]
    > 回答内容は匿名化されて分析に使用されます。

### 📤 3.2 フォーム申請
現場の気づきを元に、新しいフォーム実施を管理者に提案できます。
※ ボトムアップ申請機能です。

1.  ダッシュボードの `フォーム申請` タブを選択し、**`フォーム申請作成`** ボタンをクリックします。
2.  タイトル、説明、独自の設問を作成します。
3.  **`申請`** ボタンを押します。
    > [!IMPORTANT]
    > 申請後は自動的にチャット画面（編集画面）に遷移します。ここで管理者と追加の連絡を取ることができます。

#### ステータスの種類
- **申請中**: 管理者が確認中の状態です。
- **承認済み**: 管理者が承認した状態です。その後、管理者が「公開」操作を行うと、全メンバーが回答可能になります。
- **却下**: 修正が必要な場合など。修正して再申請が可能です。
- **承認済み（停止中）**: 管理者側でのステータス表記です。

## 🏢 4. 組織管理者
各組織（部署・チーム）の管理権限を持つユーザー向けの機能です。
> [!NOTE]
> 一般ユーザーの機能（回答、投稿など）も全て利用可能です。

### 📋 4.1 フォーム作成・管理
フォームの作成、および一般ユーザーからの申請の管理を行います。

#### 承認フロー
1.  **一覧確認**: サイドバーの `フォーム作成・管理` タブでフォーム一覧を確認します。
2.  **審査**: 「申請中」のフォームを確認し、**承認 (`Check`)** または **却下 (`X`)** を選択します。
3.  **状態変化**: 承認されたフォームは **「停止中」** となり、公開の準備が整った状態になります。

#### その他の操作
- **チャット機能**: 申請中のフォーム詳細画面から、申請者とチャットで連絡を取ることができます。
- **公開/停止**: 停止中のフォームに対し、**公開 (`Eye`)** アイコンを押すと「公開中」となり、組織内の全メンバーが回答可能になります。
- **結果のダウンロード**: 各フォームの `CSVダウンロード` ボタンから、全回答データをエクスポートできます。

### 📊 4.2 クラスタリング分析の実行
蓄積されたフォーム回答データをAI分析し、「分析セッション（レポート）」を作成します。
※ 分析セッション作成プロセスです。

1.  サイドバーの `データ分析実行` タブを選択します。
2.  **分析対象の選択**: 分析したい「フォーム」と、その中の特定の「質問」を選択します。
3.  **実行**: セッション名（任意）を入力し、**`分析を開始する`** をクリックします。
    > [!WARNING]
    > データ量によっては、分析完了まで数分かかる場合があります。
    > この処理で、**クラスタリング**、**Small Voice検出**、**課題リスト生成**が自動で行われます。
4.  **公開**: 分析完了後、内容を確認し、問題なければ **`公開する`** ボタンで一般メンバーに公開します。

### ✨ 4.3 雑談のAI分析
雑談掲示板の会話データをAIが分析し、組織の関心事や課題を抽出して、新しいフォームの作成を提案します。

1.  `雑談掲示板` 画面を開きます。
2.  （管理者のみ）画面右側に「作成フォームの提案」エリアが表示されます。
3.  **分析実行**: 対象期間を指定して **`AI分析を実行`** を行います。
4.  **提案の活用**: 分析結果として推奨されるテーマが表示されます。**`この提案でフォームを作成`** ボタンを押すと、提案内容に基づいたフォームの下書きが自動作成されます。

### 💡 4.4 議論のAI分析
分析セッション（レポート）内の課題ごとの議論チャットに対しても、AI分析を実行できます。これは「AIファシリテーター」機能として提供されます。

1.  対象の分析セッション詳細画面を開きます。
2.  画面下部の「AIファシリテーター」セクションにある **`分析を実行`** （または「再分析」）ボタンを押します。
3.  AIが議論の流れを分析し、**次のアクション** を具体的に提案します。
4.  分析結果を確認し、**`公開`** することで一般メンバーにも表示されます。

### 📥 4.5 CSVデータのインポート
外部システムや過去のアンケートデータを分析用に取り込むことができます。
サイドバーの `CSVインポート` タブから実行します。

> [!TIP]
> 1行目を質問文、2行目以降を回答データとして読み込みます。

### 👥 4.6 メンバーリストの確認
1.  サイドバーの `メンバーリスト` タブを選択します。
2.  現在組織に参加しているメンバーの一覧を確認できます。
    > [!NOTE]
    > メンバーの追加・招待（アカウント作成）は、**システム管理者**のみが実行可能です。

## 🛠 5. システム管理者
システム全体の保守・管理を行う特権ユーザー向けの機能です。
サイドバー下部の `システム管理` リンク（または `/admin/system`）から専用ダッシュボードにアクセスします。

### 🏢 5.1 組織管理
`組織管理` タブで以下の操作を行います。
- **組織一覧**: システムに登録されている全組織を確認できます。
- **新規作成**: 新しい組織（部署・プロジェクト）を作成します。
- **編集・削除**: 既存組織の名称変更や削除を行います。

### 👥 5.2 ユーザー管理
`ユーザー管理` タブで以下の操作を行います。
- **ユーザー一覧**: 全登録ユーザーを確認できます。
- **新規作成 (招待)**: アカウントを作成し、招待メールを送信します。
- **権限設定**:
    - **システム権限**: システム管理者か否かを設定します。
    - **組織への所属**: 各組織への所属有無と、組織内でのロール（一般/管理者）を設定します。
- **編集・削除**: 既存ユーザー情報の変更やアカウント削除を行います。
