# タスク: 分析エンジンの抜本的改善

## 目的
ユーザーから「修正の効果が感じられない」との指摘を受け、少数意見（Small Voice）がより鮮明に抽出・可視化され、かつ分析結果に多様性が生まれるよう、分析ロジックを根本から見直す。

## 完了背景
- HDBSCAN でノイズとして扱われていた少数意見が個別に独立して可視化されること。
- 感情分析が 0.0 に偏らず、意見の強さが適切にスコア化されること。
- レポート（Issue）において、件数の多い課題だけでなく、深刻な少数意見が明確な「リスク」として提示されること。
