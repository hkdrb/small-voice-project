# 修正内容の確認 (Walkthrough)

## 実施した変更
`test_users_list.csv` を、データベースの初期データ作成コードと完全に一致するように修正しました。

### 主な修正点
1. **一般ユーザーのマッピング修正**
   - 以前は `user10@example.com` が `一般ユーザー1` となっていましたが、正しく `ユーザー10` に戻し、全体の「ずれ」を解消しました。
2. **欠落していたユーザーの追加**
   - `admin@example.com` (パスワード: `OrgAdmin1234!`) を追加しました。
3. **表示名の統一**
   - `管理者1 (兼務)` や `システム管理者 (system)` など、実際のシステム画面で表示される名前に合わせました。
4. **所属組織の正確な記載**
   - 各ユーザーがどの組織・プロジェクトに所属しているかを、`seed_db.py` のロジックに基づき正確に反映しました。

## 確認方法
修正後の `test_users_list.csv` を開き、以下の順序で並んでいることを確認してください：
1. システム管理者 (5名)
2. 組織管理者 (1名 + admin1~6の6名)
3. 一般ユーザー (user1~10の10名)
