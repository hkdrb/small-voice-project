# 実装計画：テストユーザーリストの修正

## 1. 現状確認と分析
以下の不整合を確認しました：
- **メールアドレスのずれ**: `user10` が一般ユーザーの先頭に来ており、以降が1つずつずれていた。
- **不足ユーザー**: `database.py` で定義されている初期組織管理者 `admin@example.com` が欠落していた。
- **表示名の不一致**: `管理者1 (兼務)` などの詳細な名前が、CSVでは `組織管理者1` のように簡略化されていた。
- **所属組織の過不足**: `株式会社サンプル` への所属や、特定の案件へのアサイン状況がコードと一致していなかった。

## 2. 修正方針
- データベース初期化コード（`database.py` および `seed_db.py`）を「正」とし、CSVを完全に書き換える。
- パスワードは現状維持（コード上のデフォルト値と一致）。

## 3. 実施手順
1. `test_users_list.csv` の内容を正しいデータで上書きする。
2. 修正内容の解説を行う。
