# 実装計画: イントロダクションページの強化（画像追加 & レスポンシブ化）

## 1. 変更対象ファイル
- `frontend/src/app/introduction/page.tsx`

## 2. 変更内容
### 2.1. 画像の挿入と最適化
- `next/image` のインポート追加。
- 「ブロードリスニングとは」セクションに画像コンテナを追加。
- `max-w-2xl mx-auto` でサイズを制限し、`shadow-lg` 等で装飾。

### 2.2. モバイルレスポンシブ対応
- **ヒーローセクション**: `text-4xl sm:text-5xl md:text-7xl` 等、画面幅に応じたフォントサイズ調整。
- **カードレイアウト**: `p-5 sm:p-8` とし、モバイルではコンテンツ幅を確保。
- **機能リスト**: `hidden sm:block` でモバイル時にアイコンを隠し、左側の余白を解消。
- **テーブル/リンク**: `px-3 sm:px-6` への余白調整と、`break-all` によるURL突き抜け防止。
