# 修正内容の確認 (Walkthrough)

## 実施した変更

### 1. イントロダクションページへの画像追加
- ブロードリスニングの解説文直後に、コンセプト画像を挿入しました。
- 画像が大きすぎないよう `max-w-2xl` で制限し、中央寄せに配置しています。

### 2. モバイレスポンシブ対応の強化
- **タイトルの可変化**: ヒーローセクションのタイトル等のフォントサイズを画面幅に応じて最適化しました。
- **カードの余白調整**: モバイル表示時に余白を詰め、コンテンツを読みやすくしました。
- **機能アイコンの制御**: スマホ表示時に「主な機能」のアイコンを非表示（`hidden sm:block`）にすることで、左側の不自然な余白を解消しました。
- **テーブルとURL**: モバイルでの視認性を高めるため、テーブルの余白調整とURLの折り返し設定を行いました。
