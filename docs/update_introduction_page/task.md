# タスク: イントロダクションページへの画像挿入とレスポンシブ対応

## 概要
`/introduction` ページの「ブロードリスニングとは」の説明の後に解説用画像を挿入し、同時にページ全体のスマートフォン（モバイル）表示を最適化する。

## ステップ
1.  `/introduction` ページのソースコード (`frontend/src/app/introduction/page.tsx`) を確認。
2.  挿入箇所（「ブロードリスニングとは」の解説文の後）に `next/image` を使用して画像を挿入。
3.  画像サイズを `max-w-2xl` に制限し、中央寄せに設定。
4.  ページ全体のモバイルレスポンシブ対応を実施。
    - ヒーローセクションのタイトル、本文のフォントサイズを可変に。
    - 各カードのパディングをモバイル向けに調整 (`p-5 sm:p-8`)。
    - 主な機能セクションでモバイル時にアイコンを非表示にし、余白を最適化。
    - テーブルの余白調整およびURLの折り返し設定。
5.  変更内容をリポジトリにプッシュ。
